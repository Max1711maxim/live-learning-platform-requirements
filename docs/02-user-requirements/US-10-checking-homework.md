# US-10: Массовая проверка домашних заданий с шаблонами комментариев

**Идентификатор:** US-10

**Роли:** Преподаватель (Автор курса)

**Формулировка требования:**
Как преподаватель, я хочу открывать список всех загруженных работ студентов, фильтровать непроверенные, применять заготовленные шаблоны комментариев и массово выставлять оценки, чтобы сократить время на рутинную проверку при росте числа студентов.

**Атрибуты:**

| Атрибут | Значение |
|:---|:---|
| Источник | Impact Mapping, Use Case UC-2, Модель Кано (One-dimensional) |
| Приоритет | Высокий (One-dimensional, Релиз 2) |
| Стабильность | Стабильное |

**Связанные требования:**
- FR-05: Система должна позволять преподавателю создавать, редактировать и сохранять не менее 20 шаблонов типовых комментариев для повторного использования при проверке ДЗ.
- FR-05: При массовом применении шаблона к группе работ система должна выполнить операцию не более чем за 2 секунды для группы до 50 студентов и показать уведомление об успешном завершении или ошибке.
- NFR-01: Интерфейс предпросмотра файлов ДЗ в ленте проверки должен открывать превью работы не более чем за 3 секунды.
