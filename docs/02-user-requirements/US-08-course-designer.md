# US-08: Drag-and-Drop конструктор структуры курса

**Идентификатор:** US-08

**Роли:** Преподаватель (Автор курса)

**Формулировка требования:**
Как преподаватель, я хочу создавать и редактировать структуру курса путём перетаскивания блоков уроков, тестов и вебинаров, чтобы запускать обучение самостоятельно, без привлечения технической поддержки.

**Атрибуты:**

| Атрибут | Значение |
|:---|:---|
| Источник | Impact Mapping, Модель Кано (Must-be) |
| Приоритет | Критический (Must-be) |
| Стабильность | Стабильное |

**Связанные требования:**
- FR-10: Система должна предоставлять интерфейс Drag-and-Drop для добавления, удаления и изменения порядка модулей и уроков курса без перезагрузки страницы.
- NFR-03: Интерфейс конструктора курсов должен корректно работать на экранах ноутбуков (от 1366×768 px) и поддерживать сенсорный ввод на планшетах.
