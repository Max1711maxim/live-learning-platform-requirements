# US-04: Просмотр верифицированных отзывов

**Идентификатор:** US-04

**Роли:** Студент (Слушатель)

**Формулировка требования:**
Как студент, я хочу видеть рейтинг и отзывы только от пользователей, которые действительно оплатили и прошли курс, чтобы доверять этой информации перед принятием решения о покупке.

**Атрибуты:**

| Атрибут | Значение |
|:---|:---|
| Источник | Карта эмпатии Марии, Модель Кано (Must-be) |
| Приоритет | Критический (Must-be) |
| Стабильность | Стабильное |

**Связанные требования:**
- FR-03: Система должна проверять статус оплаты и процент завершения курса перед публикацией отзыва и отображать плашку «Пользователь прошёл курс полностью» только для верифицированных студентов.
- FR-08: Система должна отправлять уведомление преподавателю при публикации нового верифицированного отзыва.
