# US-02: Синхронизация прогресса между устройствами

**Идентификатор:** US-02

**Роли:** Студент (Слушатель)

**Формулировка требования:**
Как студент, я хочу, чтобы место остановки в видео и результаты пройденных тестов автоматически сохранялись в облаке, чтобы я мог начать обучение на ПК и продолжить с того же места на смартфоне без потери данных.

**Атрибуты:**

| Атрибут | Значение |
|:---|:---|
| Источник | Карта эмпатии Марии (Студент), Модель Кано (Must-be) |
| Приоритет | Критический (Must-be) |
| Стабильность | Стабильное |

**Связанные требования:**
- FR-04: Система должна сохранять таймкод остановки видео с точностью до секунды и автоматически начинать воспроизведение с этого места при открытии урока на любом устройстве пользователя.
- NFR-01: Время синхронизации прогресса между устройствами не должно превышать 3 секунд с момента открытия урока.
- NFR-04: Данные о прогрессе студента должны передаваться и храниться в зашифрованном виде.
