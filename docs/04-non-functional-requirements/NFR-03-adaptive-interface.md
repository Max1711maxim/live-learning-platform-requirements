# NFR-03: Адаптивный интерфейс (Mobile-first)

**Идентификатор:** NFR-03

**Формулировка требования:**
Пользовательский интерфейс платформы должен удовлетворять следующим требованиям:

1. **Адаптивная вёрстка:** интерфейс должен корректно отображаться и сохранять функциональность на устройствах с шириной экрана:
   - От 320 px (смартфоны).
   - От 768 px (планшеты).
   - От 1366 px (ноутбуки и ПК).

2. **Mobile-first:** приоритет при проектировании — сценарии использования на смартфонах (просмотр лекций, прохождение тестов, чат).

3. **Конструктор курсов:** интерфейс Drag-and-Drop должен поддерживать как мышь, так и сенсорный ввод (touch events) на планшетах.

4. **Видеоплеер:** должен адаптироваться под размер экрана без выхода элементов управления за границы.

5. **Поддержка браузеров:** последние две стабильные версии Chrome, Safari, Firefox, Edge.

**Атрибуты:**

| Атрибут | Значение |
|:---|:---|
| Источник | Карта эмпатии Марии (Студент), Impact Mapping |
| Приоритет | Высокий |
| Стабильность | Стабильное |

**Связанные требования:**
- FR-10: Drag-and-Drop конструктор курсов.
- US-01: Просмотр видеолекции.
