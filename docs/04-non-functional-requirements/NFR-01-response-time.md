# NFR-01: Время отклика интерфейса

**Идентификатор:** NFR-01

**Формулировка требования:**
Система должна обеспечивать следующее максимальное время отклика (response time) при выполнении ключевых операций:

| Операция | Максимальное время |
|:---|:---|
| Открытие страницы урока с видеоплеером | 3 секунды |
| Нажатие Play/Pause на плеере | 200 мс |
| Сохранение таймкода при паузе | 500 мс |
| Синхронизация прогресса между устройствами | 3 секунды |
| Применение шаблона комментария к группе до 50 студентов | 2 секунды |
| Открытие предпросмотра файла ДЗ | 3 секунды |
| Обновление данных на дашборде аналитики | 60 секунд |
| Доставка Push-уведомления после события | 1 минута |

**Атрибуты:**

| Атрибут | Значение |
|:---|:---|
| Источник | US-01, US-02, US-10, FR-08 |
| Приоритет | Высокий |
| Стабильность | Стабильное |

**Связанные требования:**
- FR-04: Сохранение и синхронизация таймкода.
- FR-05: Шаблоны комментариев для массовой проверки ДЗ.
- FR-08: Система Push-уведомлений.
