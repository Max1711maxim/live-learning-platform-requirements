# FR-07: Автоматическое снижение качества видео при слабом интернете

**Идентификатор:** FR-07

**Формулировка требования:**
Во время воспроизведения видео система должна в реальном времени отслеживать скорость интернет-соединения студента. При падении скорости ниже порогового значения, приводящего к буферизации (зависанию видео), система должна:
1. Автоматически снизить качество видео до 480p.
2. Если буферизация продолжается — снизить качество до 360p.
3. Отобразить индикатор «Авто-качество» на плеере.
4. При восстановлении стабильного соединения — вернуть качество видео к исходному значению.

Снижение качества должно происходить без остановки воспроизведения и без необходимости ручного вмешательства студента.

**Атрибуты:**

| Атрибут | Значение |
|:---|:---|
| Источник | US-01 (Просмотр видеолекции) |
| Приоритет | Высокий (One-dimensional) |
| Стабильность | Стабильное |

**Связанные требования:**
- US-01: Просмотр видеолекции с интерактивным чатом.
- NFR-03: Адаптивный интерфейс и работа на мобильных устройствах.
