# FR-05: Шаблоны комментариев для массовой проверки ДЗ

**Идентификатор:** FR-05

**Формулировка требования:**
Система должна предоставлять преподавателю возможность:
1. Создавать, редактировать и удалять шаблоны типовых комментариев (пресеты) в количестве не менее 20 штук.
2. При проверке домашних заданий выбирать один или несколько шаблонов из списка и применять их к работе одного или группы студентов.
3. При массовом применении шаблона к группе до 50 студентов система должна выполнить операцию присвоения комментария и оценки не более чем за 2 секунды.
4. Если в процессе массовой операции произошла ошибка (например, сбой сети), система должна откатить все незавершённые изменения и показать преподавателю уведомление «Операция не выполнена, попробуйте ещё раз» с указанием списка студентов, для которых операция не была завершена.

**Атрибуты:**

| Атрибут | Значение |
|:---|:---|
| Источник | US-10 (Массовая проверка ДЗ), Use Case UC-2 |
| Приоритет | Высокий (One-dimensional) |
| Стабильность | Стабильное |

**Связанные требования:**
- US-10: Массовая проверка домашних заданий с шаблонами комментариев.
- NFR-01: Время выполнения операции.
