# FR-03: Верификация отзывов о курсе

**Идентификатор:** FR-03

**Формулировка требования:**
Система должна допускать публикацию отзыва и рейтинга (от 1 до 5 звёзд) о курсе только при выполнении всех условий:
1. Пользователь оплатил курс (статус платежа «Подтверждён»).
2. Курс не находится в статусе «Возврат средств».
3. Пользователь завершил курс (прогресс = 100%) либо просмотрел не менее 50% материалов и подтвердил желание оставить отзыв досрочно.

Для отзывов от студентов, завершивших курс на 100%, система должна отображать плашку «Пользователь прошёл курс полностью».

При попытке оставить отзыв без выполнения условий система должна выводить сообщение: «Оставить отзыв могут только студенты, оплатившие и прошедшие курс».

**Связанные требования:**
- US-04: Просмотр верифицированных отзывов.
- FR-01: Блокировка возврата при невыполнении условий.
