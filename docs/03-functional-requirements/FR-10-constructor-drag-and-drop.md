# FR-10: Drag-and-Drop конструктор курсов

**Идентификатор:** FR-10

**Формулировка требования:**
Система должна предоставлять преподавателю интерфейс конструктора курсов со следующими возможностями:
1. Создание модулей и уроков путём перетаскивания (Drag-and-Drop) элементов из панели инструментов в рабочую область.
2. Изменение порядка модулей и уроков перетаскиванием.
3. Добавление различных типов контента в урок: видео, текст, тест, файлы для скачивания, вебинар.
4. Удаление модулей и уроков с подтверждением действия.
5. Все изменения должны сохраняться в режиме реального времени (автосохранение не реже чем раз в 30 секунд) и применяться без перезагрузки страницы.
6. Конструктор должен поддерживать сенсорный ввод на планшетах.
