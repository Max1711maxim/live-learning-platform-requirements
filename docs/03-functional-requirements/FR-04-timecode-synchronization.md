# FR-04: Сохранение и синхронизация таймкода видео

**Идентификатор:** FR-04

**Формулировка требования:**
Система должна сохранять текущий таймкод воспроизведения видео с точностью до 1 секунды при каждом из следующих событий:
1. Студент ставит видео на паузу.
2. Студент закрывает страницу урока или вкладку браузера.
3. Студент переключается на другое устройство.

При повторном открытии этого же урока на любом устройстве (ПК, смартфон, планшет) система должна начинать воспроизведение видео с сохранённого таймкода.

Сохранение таймкода должно происходить автоматически, без необходимости нажатия дополнительных кнопок.

**Атрибуты:**

| Атрибут | Значение |
|:---|:---|
| Источник | US-01 (Просмотр видеолекции), US-02 (Синхронизация прогресса) |
| Приоритет | Критический (Must-be) |
| Стабильность | Стабильное |

**Связанные требования:**
- US-01: Просмотр видеолекции с интерактивным чатом.
- US-02: Синхронизация прогресса между устройствами.
- NFR-01: Время отклика интерфейса.
