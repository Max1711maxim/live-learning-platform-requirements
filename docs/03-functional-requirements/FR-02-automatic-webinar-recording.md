# FR-02: Автоматическая запись и конвертация вебинара

**Идентификатор:** FR-02

**Формулировка требования:**
По завершении вебинара преподавателем (нажатие кнопки «Завершить эфир») система должна:
1. Сохранить полную видеозапись трансляции на сервере.
2. Автоматически запустить процесс конвертации записи в формат, совместимый с видеоплеером платформы (MP4, H.264).
3. Прикрепить сконвертированное видео к соответствующему модулю курса в качестве урока.
4. Отправить всем студентам курса Push-уведомление «Запись вебинара доступна».

Время от завершения трансляции до публикации записи в курсе не должно превышать 15 минут для вебинара длительностью до 2 часов.

Если трансляция была прервана из-за сбоя (пропал интернет у преподавателя), система должна сохранить записанную часть видео в черновики урока и уведомить преподавателя о необходимости проверить и опубликовать запись вручную.
