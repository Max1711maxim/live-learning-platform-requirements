# Обзор проекта: Платформа для онлайн-обучения

## Описание проекта
Платформа для проведения вебинаров и курсов с акцентом на «живое обучение». Преподаватели создают курсы через Drag-and-Drop конструктор, загружают материалы, проводят онлайн-занятия в прямом эфире со встроенной автозаписью и проверяют домашние задания. Студенты просматривают лекции в записи, выполняют тестовые задания, общаются с преподавателем в чате и получают сертификаты об окончании курса.

## Бизнес-цели
- Запустить конкурентоспособную платформу в нише «Live-first».
- Привлечь 100+ преподавателей и 5000+ студентов за первые 12 месяцев.
- Обеспечить средний показатель завершаемости курсов ≥ 75%.
- Достичь NPS среди студентов ≥ 40.

## Роли пользователей
- **Студент (Слушатель):** Покупает и проходит курсы, выполняет ДЗ, получает сертификаты.
- **Преподаватель (Автор курса):** Создаёт и настраивает курсы, проводит вебинары, проверяет ДЗ, просматривает аналитику.
- **Администратор платформы:** Модерирует контент, управляет пользователями, мониторит техническое состояние.

## Границы проекта
### MVP (Минимально жизнеспособный продукт)
- Drag-and-Drop конструктор курсов (FR-10)
- Встроенная вебинарная комната с автозаписью (FR-02)
- Синхронизация прогресса между устройствами (FR-04)
- Система верифицированных отзывов (FR-03)
- Гарантия возврата средств (FR-01)
- Адаптивный мобильный интерфейс (NFR-03)

### Последующие релизы
- Массовая проверка ДЗ с шаблонами (FR-05)
- Полная геймификация: streaks и лидерборды (FR-09)
- Офлайн-доступ к урокам
- Интеграция с LinkedIn для публикации сертификатов
